# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

Please report all security vulnerabilities
[on GitHub](https://github.com/benwr/origin_check/security),
or to root@benwr.net
